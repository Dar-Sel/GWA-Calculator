# GWA-Calculator

## Project Overview
The GWA Calculator is a Python program that computes a Grade 7 student's General Weighted Average (GWA) based on the PSHS 2025 curriculum.

It also includes a projection feature that allows users to see how improving their lowest grade can affect their overall academic standing.

## Features
- Input grades for 8 subjects
- Calculates weighted GWA
- Displays academic standing
- Identifies highest and lowest grades
- Projection tool for grade improvement
- Handles invalid input

## Subjects Included
1. Mathematics 1
2. Integrated Science
3. English 1
4. Filipino 1
5. Social Science 1
6. Computer Science 1
7. AdTech 1 (Arts, Design, Technology)
8. P.E.H.M (Physical Education, Health, Music)

## Grading System
- Math and Science: 1.7 weight
- English: 1.3 weight
- Others: 1.0 weight
- Total units: 9.7

----- GWA SUMMARY -----
With a GWA of: 1.45
You are: Very Good and Included in the Director's List

Highest Grade: 1.00
Lowest Grade: 2.25

----- PROJECTED GRADES -----
Projected Result: Raising Mathematics 1 by 0.50 points to 1.75,
bringing your GWA to 1.30.

## How It Works
- Each subject grade is multiplied by its weight
- All weighted grades are added
- The total is divided by 9.7
- The result is truncated to 2 decimal places
- The program determines the academic standing

## Notes
- Valid grade range: 1.00 to 5.00
- Lower numbers mean better grades
- Projection improves the lowest grade

## Purpose
This project was created to practice Python basics such as loops, functions, and conditionals, and to apply a real grading system.
